- **Driver Dashboard** - Accept/reject bookings, manage rides

## How to Use

### For Patients:
1. Sign up or login
2. Click "Book Ambulance" or use SOS button
3. Enter patient details and location
4. Choose ambulance type and hospital
5. Confirm booking
6. Track ambulance in real-time
7. Rate the service

### For Drivers:
1. Login to driver dashboard
2. Toggle availability status
3. Receive booking requests
4. Accept or reject requests
5. Navigate to pickup location
6. Update ride status
7. Complete trip

### For Admin:
1. Login to admin dashboard
2. Monitor all bookings and revenue
3. View analytics
4. Manage ambulances and drivers

## How It Works

1. **Book:** Patient requests ambulance with location
2. **Confirm:** Driver accepts booking
3. **Track:** Real-time GPS tracking
4. **Reach:** Ambulance arrives at hospital
5. **Rate:** Patient rates service
